WittBot 隱私條款 (Privacy Policy)

最後更新日期： 2026年3月2日

-----------------------------------------------------
感謝您使用 WittBot（以下簡稱「本機器人」）。我們非常重視您的隱私，本文件旨在說明本機器人如何處理、收集及保護您的資訊。
-----------------------------------------------------
1. 我們收集的資料

本機器人僅會收集運行功能所必需的最小化資料：

使用者識別碼 (User ID)： 用於識別使用者身分以執行特定指令。

伺服器識別碼 (Guild ID)： 用於儲存特定伺服器的設定（如自定義前綴、頻道設定）。

訊息內容： 僅在觸發指令或特定監聽功能時，暫時處理訊息文字，我們不會持久化儲存您的私聊或一般對話紀錄。
-----------------------------------------------------
2. 資料的使用方式
3. 
收集到的資料僅用於以下用途：

確保本機器人的功能正常運作（如身分組管理、指令回應）。

提供個人化的設定服務。

除法律要求或 Discord 官方要求外，我們絕不會將您的資料出售或分享給第三方。
-----------------------------------------------------
3. 資料儲存與安全
4. 
儲存： 除非功能需要（例如等級系統或設定檔），否則大部分資料皆為記憶體內處理，不進行長期存儲。

安全性： 我們使用安全的 API 傳輸協議。作為開源專案，您可以隨時在 GitHub 檢查原始碼以確認資料處理邏輯。
-----------------------------------------------------
4. 使用者權利
5. 
如果您希望刪除本機器人儲存的關於您的任何資料，您可以：

從伺服器中移除 (Kick/Ban) 本機器人。

聯絡開發者 [你的聯絡方式/Discord ID] 要求手動刪除相關資料庫記錄。
-----------------------------------------------------
5. 條款變更
我們保留隨時修改本條款的權利。重大變更將透過 GitHub 更新或在 Discord 官方支援伺服器（如有）中公佈。
